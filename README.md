# ai-portfolio-site

> 이승윤의 Machine Learning Engineer 포트폴리오 웹사이트입니다.  
> AI/ML 프로젝트, 기술 스택, 성장 과정, 수상 경험, 연락처를 한 페이지에서 확인할 수 있도록 구성한 **반응형 개인 포트폴리오 사이트**입니다.

---

## 📌 프로젝트 소개

이 프로젝트는 제가 수행한 AI/ML 프로젝트와 기술 역량을 보다 직관적으로 보여주기 위해 제작한 개인 포트폴리오 웹사이트입니다.

단순히 이력만 나열하는 형태가 아니라,  
**데이터 → 모델링 → 서비스 구현**까지 이어지는 저의 경험을  
웹사이트 형태로 정리해 한눈에 볼 수 있도록 구성했습니다.

사이트는 다크 테마 기반의 감성적인 UI로 제작했으며,  
프로젝트별 핵심 내용, 기술 스택, 성장 과정, 수상 내역, 연락처까지  
채용 담당자나 면접관이 빠르게 파악할 수 있도록 설계했습니다.

---

## 🚀 주요 기능

- **반응형 포트폴리오 웹사이트**
  - 데스크톱, 태블릿, 모바일 환경에 맞춰 레이아웃이 반응형으로 동작합니다.

- **싱글 페이지 구조**
  - About, Projects, Skills, Journey, Awards, Contact 섹션을 한 페이지에서 탐색할 수 있습니다.

- **프로젝트 중심 구성**
  - 대표 프로젝트들을 카드 형태로 정리하고, 핵심 역할과 기술 스택을 강조했습니다.

- **다크 테마 UI**
  - 다크 배경과 그라데이션, 글래스모피즘 스타일을 활용해 깔끔하고 몰입감 있는 디자인을 적용했습니다.

- **애니메이션 효과**
  - 스크롤 리빌, 진행 바, 배경 오브, 타이핑 효과, 버튼 인터랙션 등 동적인 UI를 포함했습니다.

- **모바일 네비게이션 지원**
  - 모바일 화면에서는 드로어 메뉴를 통해 섹션 이동이 가능합니다.

- **모달 기반 상세 보기**
  - 프로젝트 상세 내용을 모달 형태로 확장해 보여줄 수 있도록 구성했습니다.

## 🧠 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript
- **Design**: Dark UI, Glassmorphism, Gradient, Responsive Web
- **Font**: Google Fonts (Pretendard)
- **Deployment**: GitHub Pages (또는 정적 호스팅 가능)

## 🖥️ 주요 섹션 구성

### 1. Hero
- 포트폴리오의 첫 인상 영역
- 이름, 직무 방향성, 한 줄 소개, GitHub 링크, 프로젝트 바로가기 버튼 포함

### 2. About
- AI/ML 엔지니어로서의 방향성과 강점 소개
- 데이터에서 모델을 만들고 서비스까지 연결하는 경험 강조

### 3. Projects
- 주요 프로젝트를 카드 형태로 정리
- 사용 기술, 핵심 역할, 프로젝트 특징을 한눈에 볼 수 있도록 구성

### 4. Skills
- 보유 기술 스택 시각화
- AI/ML, 데이터 처리, 서비스 개발 관련 기술을 정리

### 5. Journey
- 성장 과정과 학습 흐름
- 어떻게 현재의 역량까지 발전했는지 타임라인 형태로 전달

### 6. Awards
- 수상 및 활동 경험 정리
- 프로젝트 외의 성과도 함께 보여줄 수 있도록 구성

### 7. Contact
- GitHub, 이메일 등 연락 가능한 정보 정리
- 협업 및 채용 문의를 위한 마무리 영역

## ✨ 구현 포인트

### 1. 단일 HTML 파일 기반 포트폴리오
별도의 프레임워크 없이도 충분히 완성도 있는 포트폴리오를 만들 수 있도록  
HTML, CSS, JavaScript만으로 전체 사이트를 구현했습니다.

### 2. 인터랙티브 UI 구성
정적인 이력서 느낌이 아니라,  
사용자가 직접 스크롤하고 탐색하면서 자연스럽게 저의 경험을 이해할 수 있도록  
스크롤 애니메이션, 모달, 타이핑 효과 등을 넣었습니다.

### 3. 프로젝트 중심 정보 구조
단순 소개가 아니라,  
실제로 수행한 프로젝트와 역할, 기술 역량이 중심이 되도록 설계했습니다.

### 4. 채용/포트폴리오 목적 최적화
면접관이나 채용 담당자가 빠르게 핵심 정보를 파악할 수 있도록  
섹션별 정보를 압축적으로 보여주되, 시각적으로는 깔끔하게 정리했습니다.

## 🛠️ 실행 방법

별도의 빌드 과정 없이 `index.html` 파일만 열면 실행할 수 있습니다.

### 로컬에서 실행
```bash
# 저장소 클론
git clone https://github.com/leeseungyoons/ai-portfolio-site.git

# 폴더 이동
cd ai-portfolio-site

# index.html 실행
```

브라우저에서 `index.html` 파일을 직접 열거나,  
VS Code의 Live Server 확장을 사용하면 더 편하게 확인할 수 있습니다.

제가 어떤 문제를 어떻게 해결해 왔는지,
