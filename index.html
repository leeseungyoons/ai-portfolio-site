센서 데이터 · 분류 모델

가속도/자이로 등 스마트폰 센서 데이터로 사용자의 행동을 분류하는 모델.

Sensor HAR
  • 센서 시계열 데이터를 윈도우 단위로 분할하고 통계·주파수 기반 특징 추출
  • 여러 분류 모델을 비교하며 행동 인식 정확도 향상을 위한 실험 수행
  • 혼동행렬을 활용해 오분류 패턴 분석 및 개선 방향 도출
AI 면접관 Agent (이력서·자소서 기반 면접 Q/A·평가·리포트 자동화)
LLM · LangGraph · 인터뷰 시뮬레이터

이력서·자기소개서를 기반으로 면접 질문을 생성하고, 답변을 평가한 뒤 인터뷰 리포트를 자동으로 작성하는 AI 면접관 에이전트.

Agent Interview
  • InterviewState로 이력서 요약, 키워드, 질문 전략, Q/A 로그, 평가 결과를 구조화
  • 이력서 분석 → 질문 생성 → 답변 평가 → 추가 질문 여부 결정 → 리포트 작성 플로우 설계
  • LangGraph로 각 단계를 노드로 구성해 규칙 기반 로직과 LLM을 조합한 인터뷰 경험 구현
AI 강사 Agent (강의자료 → 수업 설계·퀴즈·피드백 자동화)
LLM · 교육 도메인 · 콘텐츠 생성

강의자료를 입력하면 수업 구성, 핵심 요약, 퀴즈, 과제, 피드백까지 자동으로 생성하는 교육용 AI 강사 에이전트.

Education Content Gen
  • 강의자료에서 챕터별 핵심 개념을 추출하고, 학습 목표와 수업 흐름을 자동 설계
  • 객관식·주관식 퀴즈와 과제를 생성해 학습 내용을 점검할 수 있도록 구성
  • 예시 답변 기반 피드백 문장을 생성해 “강의자료 → 수업 → 평가 → 피드백” 플로우를 자동화
