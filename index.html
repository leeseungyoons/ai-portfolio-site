AI 면접관 Agent (이력서·자소서 기반 면접 Q/A·평가·리포트 자동화)
LLM · LangGraph · 인터뷰 시뮬레이터

지원자의 이력서·자기소개서를 기반으로 면접 질문을 생성하고, 답변을 평가한 뒤 인터뷰 리포트를 자동으로 작성하는 AI 면접관 에이전트.

  • InterviewState를 정의해 이력서 요약, 키워드, 질문 전략, Q/A 로그, 평가 결과를 상태로 관리
  • 이력서 분석 → 질문 생성 → 답변 평가(관련성·구체성·논리성 등) → 추가 질문 여부 결정 → 최종 리포트 작성 플로우 설계
  • LangGraph로 각 단계를 노드로 구성하고, 규칙 기반 로직과 LLM을 조합해 일관된 인터뷰 경험 구현
