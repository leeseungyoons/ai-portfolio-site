About

데이터 기반 문제를 서비스까지 연결하는 AI/ML 엔지니어 지향

단순히 모델 성능 수치만 올리는 것이 아니라, 결과가 실제 의사결정과 사용자 경험에 쓰이도록 구조화하는 데 강점을 두고 있습니다. NLP, 시계열 예측, 센서 데이터 분류, LLM Agent, 웹/백엔드 연동까지 폭넓게 경험했습니다.

강남대학교 인공지능융합공학부 인공지능 전공으로, 다양한 수업과 프로젝트를 통해 AI 모델링 + 서비스 개발 + 팀 리딩 경험을 쌓았습니다. 초기에는 Python/배포도 낯설었지만, VS Code와 Streamlit부터 시작해 현재는 AWS/CI-CD/웹 연동까지 확장하며 성장했습니다.

특히 KT AIVLE 프로젝트에서는 K-Food 수출·현지화 플랫폼에서 수요/트렌드 예측, ABSA 기반 VOC 정량화, JSON 파이프라인 표준화, 도움말 챗봇 웹 연결 등 “데이터 → 예측 → 리포트 → 서비스” 전체 흐름을 경험했습니다.

Journey

초기 Python/배포 초보에서 AI 프로젝트를 서비스로 연결하는 단계까지

단순 기술 나열보다, 어떻게 성장했는지를 보여주면 면접에서 훨씬 강해집니다. 아래 흐름은 자기소개/면접 답변에도 그대로 사용할 수 있는 구조입니다.

STEP 01 · 학부 초중반
Python, AI 기초 수업 기반으로 문제 해결 시작
인공지능입문·수학·코딩테스트기초 등 기초 과목을 통해 Python과 알고리즘, AI 개념을 익히며 실제 구현 중심으로 감을 잡기 시작.
STEP 02 · 프로젝트 확장
게임/앱/분석 프로젝트로 구현 범위 넓히기
미로+퀴즈+스도쿠 게임(학술제 수상), React Native AI 작가 앱, 맛집 추천 서비스 등으로 단순 모델링을 넘어 UI/UX와 서비스 흐름까지 경험.
STEP 03 · ML/NLP/시계열 실험
스포츠 기사 감성분석, 풍력 발전량 예측 등 모델링 역량 강화
텍스트 분류, 시계열 피처 엔지니어링, 모델 비교 실험을 통해 데이터 기반 문제 해결 감각을 강화.
STEP 04 · 연구/공모전/학술 성과
논문 우수상·공모전 등 결과물로 증명
화재 피난 경로 탐색 연구(학생 논문 우수상), 풍력 발전량 예측 공모전, 공공데이터 기반 서비스 설계 등 다양한 도메인에서 문제 해결 경험 축적.
STEP 05 · KT AIVLE Big Project
AI 모델 결과를 실제 비즈니스 리포트/웹 기능으로 연결
K-Food 수요/트렌드 예측, ABSA VOC 정량화, AI 리포트/도움말 챗봇, Agent 워크플로우 등으로 데이터-모델-서비스를 하나의 파이프라인으로 연결.
